## Project setup :hammer:
#### Setup project:
```
npm install
```
#### Compiles and hot-reloads for development
```
npm run serve
```
#### Compiles and minifies for production
```
npm run build
```
&nbsp;
## Run with Docker compose :whale:
```
docker-compose up
```
